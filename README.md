# Programske paradigme - materijali za samostalni rad

**Napomena**: Repozitorijum je trenutno u izradi.

Komplementarni materijali za zbirku zadataka za kurs _Programske paradigme_ na smeru Informatika, Matematičkog fakulteta u Beogradu. 
U okviru ovog repozitorijuma se nalaze rešenja zadataka iz zbirke, po oblastima:
- [Komponentno programiranje](./component), jezik _Python_ uz radni okvir [PyQt](https://doc.qt.io/archives/qtforpython-5/) verzije 5
- [Programiranje ograničenja](./constraint), jezik _Python_ uz radni okvir [Python Constraint](https://pypi.org/project/python-constraint/), jezik _Prolog_
- [Funkcionalno programiranje](./functional), jezik _Haskell_
- [Konkurentno programiranje](./concurrent), jezik _Scala_
- [Distribuirano programiranje](./distributed), jezik _Scala_, uz radni okvir [_Apache Spark_](https://spark.apache.org/)
- [Logičko programiranje](./logical), jezik _Prolog_
